# CAIRN

An opinionated verification stack for AI-driven coding on the bnapier.dev standard stack: Vite+, React 19, TypeScript strict, Convex, Better Auth, Vercel.

A reference pattern (docs + configs) any new app can adopt to get four layers of enforcement — instructions, in-loop hooks, local git gates, and authoritative CI/branch protection — without re-deriving the scaffolding each time.

See [PRD.md](PRD.md) for the full specification.
